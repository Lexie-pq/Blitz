🌟 Project Overview
Blitz is a next-generation smart companion application designed exclusively for BESV, a premium electric bicycle brand known for its philosophy of Beauty, Eco-friendly, Smarter, and Vision.

This project bridges the gap between high-end hardware and digital lifestyle. It is a high-fidelity prototype built with Figma, featuring a live-connected database with mock data to simulate real-world operations such as cycling statistics, community interaction, and eco-gamification.

🚀 Key Features
Smart Dashboard (Main Page): Real-time monitoring of BESV bike status (battery, mileage) integrated with live GPS navigation.

City Check-in: A culture-tourism integration feature that encourages users to explore urban landmarks and earn rewards.

Carbon Footprint Ranking: Gamified environmental impact tracking that calculates carbon savings based on distance and speed, awarding exclusive badges.

Green Energy Exchange: Converts physical effort (calories/climbing height) into virtual "Green Energy Points" to redeem battery packs or support public welfare projects.

Supply Stations: Context-aware discovery of nearby maintenance and rest points.

Community Hub: A social platform for riders to connect based on location and shared interests.

🛠️ How it was Built
Prototyping: Built entirely in Figma with advanced components and interactive states.

Database Integration: Connected to a functional backend database containing mock data, allowing for realistic data visualization and user interaction.

AI-Enhanced Production: To demonstrate the flexible use of cutting-edge AI tools, the software's promotional video was co-created using OpenAI's Sora, showcasing advanced visualization of the app's vision.

🌟 项目简介
Blitz 是一款专为高端电动自行车品牌 BESV 打造的智能伴侣应用。BESV 品牌以“美丽 (Beauty)、环保 (Eco-friendly)、智能 (Smarter) 及愿景 (Vision)”为核心理念，而 Blitz 则是这一理念在数字化领域的延伸。

本项目是一个利用 Figma 制作的高保真原型，不仅实现了精美的 UI 交互，还成功连接了数据库（包含 Mock 模拟数据），支持用户进行基本的查看与交互操作，如查看骑行统计、参与社区互动及绿色能源兑换。

🚀 核心功能
智能仪表盘 (主页)： 实时显示 BESV 自行车的硬件状态（电量、里程等），并集成地图导航功能。

城市打卡 (文旅结合)： 鼓励用户骑行至城市著名景点进行打卡，解锁相应实物或虚拟奖励。

碳足迹排名： 基于骑行距离与速度计算碳减排量，并设计了动态排名与徽章系统，激励绿色出行。

绿能收集与兑换： 将运动产生的能量值（基于距离与爬升高度计算）兑换为电池包，或捐赠至公益能源项目。

驿站补给： 智能展示骑行路线附近的补给站及物资储备情况。

社交社区： 为用户提供交流平台，支持基于地理位置或共同兴趣添加好友，构建骑行生态。

🛠️ 开发工具与技术
原型设计： 使用 Figma 进行深度开发，包含复杂的组件逻辑与交互链路。

数据驱动： 页面连接至数据库，利用 Mock 数据模拟真实的业务流。

AI 工具赋能： 本项目的软件宣传视频采用了 Sora 辅助制作。通过灵活运用前沿 AI 工具，将软件的愿景以极具冲击力的视觉效果呈现出来，体现了对新技术与设计流程的深度整合能力。
